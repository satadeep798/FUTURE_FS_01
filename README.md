# Deep's Portfolio

A modern, professional, and highly animated developer portfolio built with React, Three.js, and Framer Motion.

## 🚀 Features

- **3D Interactive Background**: Immersive particle field powered by Three.js and `@react-three/fiber`.
- **Bento Grid Layout**: Modern UI design for About and Skills sections.
- **Smooth Scrolling**: Integrated with Lenis for a premium feel.
- **Interactive Project Cards**: 3D tilt effects and hover states.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Contact Form**: Integrated backend for handling user inquiries.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS 4, Framer Motion (motion).
- **3D/Graphics**: Three.js, `@react-three/fiber`, `@react-three/drei`.
- **Backend**: Node.js, Express (configured for Vercel Serverless Functions).
- **Deployment**: Optimized for Vercel.

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/satadeep798/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
